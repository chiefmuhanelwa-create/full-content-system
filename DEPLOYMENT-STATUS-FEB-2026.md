## 🐛 Known Issues & Solutions

### Issue 1: API Key Not Set

**Symptom**: "ANTHROPIC_API_KEY environment variable is not set"

**Solution**:
1. Go to platform dashboard (Vercel/Netlify)
2. Settings → Environment Variables
3. Add `ANTHROPIC_API_KEY` with your key
4. Redeploy application

### Issue 2: Functions Timeout

**Symptom**: Script generation fails with timeout

**Solutions**:
- **Vercel Free**: Upgrade to Pro (60s timeout) OR
- **Switch to Netlify**: 26s timeout (better for AI)

### Issue 3: Build Fails

**Symptom**: Build errors during deployment

**Solutions**:
1. Check Node.js version (must be 18+)
2. Run `npm run build` locally first
3. Check for TypeScript errors
4. Verify all dependencies installed

### Issue 4: Scripts Not Generated

**Symptom**: Generate button doesn't work

**Check**:
1. Browser console for errors (F12)
2. Network tab for API call status
3. Verify ANTHROPIC_API_KEY is set
4. Check API key is valid

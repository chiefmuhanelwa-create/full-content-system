# 🚀 NOCHILL - Viral Content Creation System

Open-source AI-powered viral content creation platform using Claude. Free forever.

## 🌐 Live Demo

**Production URL:** https://full-content-system-nochill.vercel.app/

## ✨ Features

### 🎯 New Business System Features
- **💰 Sales Script Mode** - 10-step storytelling framework for product selling
- **🗂️ Product Database** - Full CRUD interface with 4 pre-loaded products
- **🏷️ Audience Level Vault** - 75+ content ideas & stories categorized by creator level

### 📝 Core Content Tools
- **🎣 Hook Generator** - R×A×C×U^B formula for scroll-stopping hooks
- **📖 Script Writer** - NOCHILL 5-Line Method with Ubuntu Story Arc
- **📚 Story & Content Vault** - 110+ ideas and story variations
- **📅 Visual Calendar** - Color-coded content planning grid
- And 18+ more content creation tools...

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Add your API key to .env
ANTHROPIC_API_KEY=your_api_key_here

# Run development server
npm run dev
```

## 🌐 Deploy to Vercel

1. Push your code to GitHub
2. Import project in Vercel
3. Add environment variable: `ANTHROPIC_API_KEY`
4. Deploy
